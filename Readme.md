## My process

1) Clone the project

   ```https://github.com/MohamedEldefrawy/E-commerce-Webapp.git```
2) in the project directory run the following
    1) schema.sql script
    2) data.sql script
3) ```
    mvn install
   ```

4) Deploy the War file in tomcat
